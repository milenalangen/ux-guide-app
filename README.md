# UX Design Productivity App

This is a code bundle for UX Design Productivity App. The original project is available at https://www.figma.com/design/bGH3cb2BQ613cyMSiKZU6B/UX-Design-Productivity-App--Copy-.

The project is deployed with Netlify and available on https://ux-guide-app.netlify.app/

## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
